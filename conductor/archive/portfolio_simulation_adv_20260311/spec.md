# Specification: Portfolio Simulation & Management (Phase 2)

## Overview
Phase 1에서 구축한 단위 기반 자산 관리 엔진을 확장하여, 실제 투자 성과를 시뮬레이션하고 정량적 리스크를 분석하는 고급 기능을 구현합니다. 백테스팅, 리밸런싱 가이드, 통계적 리스크 지표를 통해 사용자의 합리적인 투자 의사결정을 지원합니다.

## Objectives
- **백테스팅 엔진 구축**: 최근 2년 데이터를 기반으로 적립식(DCA) 및 거치식 투자 성과를 시뮬레이션합니다.
- **리밸런싱 최적화**: 목표 비중 대비 현재 비중의 괴리를 계산하여 매매 가이드를 제공합니다.
- **정량적 리스크 분석**: MDD, 샤프 지수, 베타 등 통계적 지표를 통해 포트폴리오의 위험도를 평가합니다.
- **상관관계 시각화**: 종목 간 상관계수를 분석하여 포트폴리오의 분산 효과를 검증합니다.

## Functional Requirements
### 1. 백테스팅 엔진 (89-90)
- **전략 지원**: 매월 정액 적립식(DCA) 투자 및 일시불 거치식 투자 시뮬레이션.
- **기간 설정**: 최근 2년(Fast mode) 고정 시세 데이터 활용.
- **벤치마크 비교**: 국내 지수(KOSPI/KOSDAQ) 수익률과 포트폴리오 수익률을 동일 차트에서 비교.

### 2. 리밸런싱 계산기 (88)
- **목표 비중 설정**: 포트폴리오 내 각 종목별 목표 비중(%) 입력 기능.
- **괴리율 계산**: 현재 시장 가치 기준 비중과 목표 비중의 차이 산출.
- **주문 가이드**: 목표 비중에 도달하기 위해 필요한 종목별 가감 수량(Buy/Sell) 추천.

### 3. 고급 리스크 지표 (91-93)
- **MDD (최대 낙폭)**: 시뮬레이션 기간 중 발생한 최대 하락폭 계산.
- **Sharpe Ratio (위험 대비 수익률)**: 변동성 대비 초과 수익률 평가.
- **Beta (시장 민감도)**: 국내 지수 변동 대비 포트폴리오의 민감도 분석.
- **Batch 기반 산출**: 지표의 계산 부하를 고려하여 일 단위 배치 작업을 통해 미리 계산 및 저장.

### 4. 상관관계 분석 (94)
- **상관계수 행렬**: 종목 간 가격 변동 유사성을 -1 ~ 1 사이의 값으로 계산.
- **분산 효과 리포트**: 상관관계가 높은 종목군을 식별하여 포트폴리오 집중 리스크 경고.

## Non-Functional Requirements
- **정밀도**: 모든 금융 계산은 `BigDecimal`을 사용하며, 퍼센트 단위는 소수점 4자리까지 보장.
- **응답성**: 복잡한 통계 계산은 배치로 처리하여 API 조회 시 지연 시간을 최소화.
- **확장성**: 향후 해외 지수(S&P 500 등) 확장이 용이한 구조 설계.

## Acceptance Criteria
- [ ] 적립식/거치식 시뮬레이션 결과가 차트 데이터로 반환된다.
- [ ] 리밸런싱 버튼 클릭 시 현재가 기준 필요 매매 수량이 정확히 계산된다.
- [ ] 배치 작업 완료 후 포트폴리오 상세 페이지에서 MDD, 샤프, 베타 지수가 표시된다.
- [ ] 종목 간 상관관계 데이터를 통해 히트맵용 데이터를 제공한다.
