# Specification: 와이어프레임 기반 포트폴리오 및 백테스트 API 기능 완성 (이슈 #146)

## Overview
와이어프레임(`WF_Portfolio`, `WF_HealthDiagnosis`, `WF_BacktestResult`)의 시각적 구성 요소를 실제 데이터로 구동하기 위한 API 고도화 및 데이터 정밀도를 향상시키는 작업입니다.

## Functional Requirements
1. **백테스트 성과 지표 산출**:
   - 백테스트 API 호출 시 상세 성과 지표(CAGR, MDD, Sharpe Ratio)를 산출하여 반환합니다.
2. **건강 진단 데이터 정밀화**:
   - 레이더 차트용 5대 차원(수익, 안전, 분산, 민첩, 현금) 데이터를 정밀하게 계산하는 로직을 추가합니다.
3. **섹터 및 종목 상세 데이터 완결성**:
   - 섹터 대시보드 및 종목 상세 조회 시 AI 인사이트와 기술적 지표 데이터가 누락 없이 반환되도록 보완합니다.

## Acceptance Criteria
- [ ] 백테스트 API 응답에 CAGR, MDD, Sharpe 지표가 정확히 계산되어 포함되어야 합니다.
- [ ] 건강 진단 API가 5대 차원에 대한 정확한 0-100 사이의 점수를 반환해야 합니다.
- [ ] 섹터 및 종목 상세 API에 누락된 AI 조언 및 지표가 포함되어야 합니다.
- [ ] 기능 추가 및 수정으로 인해 기존 테스트가 깨지지 않아야 하며, 추가된 로직에 대한 테스트 커버리지를 만족해야 합니다.
