# Product Guide: Stockwellness Backend Server

## 1. Initial Concept
개인 투자자를 위한 자산 배분 시뮬레이터 및 AI 기반 투자 분석 서비스.
감정적 거래를 데이터 기반 의사결정으로 대체하는 것이 핵심 가치입니다.

## 2. Target Audience
*   **개인 투자자:** 스스로 자산 배분 포트폴리오를 구성하고 체계적으로 관리하고자 하는 주도적인 투자자.
*   **데이터 지향 투자자:** 직관이나 감정이 아닌, 과거 데이터의 백테스트 결과와 분석 지표를 기반으로 의사결정을 내리려는 사용자.
*   **투자 학습자:** 다양한 자산 배분 전략을 시뮬레이션해 보며 시장의 흐름과 투자 원칙을 학습하고자 하는 사용자.

## 3. Primary Goals
*   **감정적 거래 지양:** 객관적인 데이터와 AI 분석을 통해 시장 변동성에 흔들리지 않는 이성적인 투자 환경 조성.
*   **포트폴리오 최적화:** AI 어드바이저와 과거 시장 데이터를 활용하여 사용자의 목표 수익률과 위험 성향에 맞는 최적의 자산 배분 비율 제시.
*   **건전성 진단:** MDD(최대 낙폭), Sharpe Ratio(위험 조정 수익률), Beta 등 전문적인 금융 지표를 통해 현재 포트폴리오의 건강 상태를 명확하게 진단.

## 4. Key Features
*   **백테스트 시뮬레이터:** 거치식 투자 및 적립식 투자(DCA) 등 다양한 시나리오에 대한 과거 성과를 시뮬레이션하여 전략의 타당성 검증.
*   **AI 리밸런싱 어드바이저:** OpenAI의 언어 모델을 활용하여 현재 시장 상황과 포트폴리오 상태를 종합적으로 분석하고, 개인화된 리밸런싱 조언 제공.
*   **분산 투자 분석:** 자산군, 산업 섹터, 국가별 투자 비중을 시각화하고 종목 간 상관관계 행렬을 제공하여 진정한 의미의 분산 투자가 이루어지고 있는지 분석.

## 5. UX Principles
*   **데이터 시각화:** 복잡하고 방대한 금융 데이터를 사용자가 직관적으로 이해할 수 있도록 명확한 차트와 그래프로 단순화하여 제공.
*   **실행 가능한 인사이트:** 단순한 수치나 통계의 나열에 그치지 않고, 사용자가 즉각적으로 포트폴리오 조정에 활용할 수 있는 구체적인 가이드와 인사이트 제공.
*   **데이터 신뢰성:** KIS API 등 검증된 외부 데이터를 기반으로 정확하고 지연 없는 금융 정보를 제공하여 서비스 전반에 대한 높은 신뢰도 구축.
